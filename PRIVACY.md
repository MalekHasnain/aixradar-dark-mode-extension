# Privacy Policy — Dark Mode Toggle (Night Eye)

*Last updated: July 5, 2026*

## Overview

Dark Mode Toggle ("the extension") is committed to protecting your privacy. This privacy policy explains what data we collect, how we use it, and your rights.

## Data Collection

**We do NOT collect any personal data.**

The extension:
- Does **not** collect, store, or transmit any personal information
- Does **not** track your browsing history
- Does **not** use analytics or tracking pixels
- Does **not** require any account or login
- Does **not** communicate with any external server

## Local Storage

All extension settings are stored **locally** in your browser using Chrome's `chrome.storage.local` API. This includes:
- Dark mode on/off preference
- Brightness, contrast, and sepia settings
- Site-specific preferences

This data never leaves your browser and is only accessible by the extension itself.

## Permissions

The extension requests the following permissions:
- **storage** — To save your dark mode preferences locally
- **activeTab** — To apply dark mode to the current tab you're viewing
- **scripting** — To inject dark mode CSS into web pages

No permission is used to access, collect, or transmit user data.

## Third-Party Services

The extension does **not** integrate with any third-party services, APIs, or trackers.

## Data Deletion

Uninstalling the extension automatically removes all locally stored settings.

Any changes to this privacy policy will be posted on this page with an updated date.
